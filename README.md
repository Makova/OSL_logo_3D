# OSL_logo_3D
Archivos para la impresión en 3D del logotipo de la [Oficina de Software Libre de la Universidad de Granada](http://osl.ugr.es).


Formatos disponibles:

- Alta poligonización
  logo_OSL.amf
  logo_OSL.stl
  logo_OSL.gcode

- Baja poligonización
  logo_OSL_low_poligon.amf
  logo_OSL_low_poligon.stl
